# Mbae Linus Mutugi - Portfolio

Personal portfolio website showcasing the work of Mbae Linus Mutugi, a Software Engineer, Full Stack Developer, and Graphic Designer based in Nairobi, Kenya.

## About

This is a responsive, multi-page portfolio website featuring:

- **Home** - Introduction and overview with core skills preview
- **About** - Detailed personal biography and professional summary
- **Skills** - Technical expertise and competencies
- **Experience** - Professional work history
- **Education** - Academic background
- **Projects** - Featured project portfolio
- **Contact** - Contact form and communication options

## Live Demo

Available at: [linuscreatives.com](https://linuscreatives.com)

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (jQuery), Bootstrap 5
- **Styling**: Custom CSS, Font Awesome Icons, Google Fonts (Merriweather & Merriweather Sans)
- **Features**: Smooth scrolling, responsive navigation, page loader, floating action button
